# Luxe Coin Flip

Premium white + gold single-page coin-flip webapp for Vercel.

## Setup

1) Put your images here:

- `public/images/heads.png`
- `public/images/tails.png`

2) Install & run:

```bash
npm install
npm run dev
```

## Deploy (Vercel)

- Import the repo in Vercel
- Framework preset: Next.js
- Build command: `npm run build`
- Output: default

## React Bits

This project includes small, React Bits-style UI primitives (blur text reveal + animated border).
If you prefer the official React Bits components, the React Bits repo supports installing components via shadcn, e.g.:

```bash
npx shadcn@latest add @react-bits/BlurText-TS-TW
```
